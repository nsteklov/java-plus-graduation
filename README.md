# Сервисы

## Сервис статистики

Функционал сервиса статистики содержит:
- запись информации о том, что был обработан запрос к эндпоинту API;
- предоставление статистики за выбранные даты по выбранному эндпоинту.

 Спецификация сервиса статистики содержится здесь: [ewm-stats-service.json](https://raw.githubusercontent.com/yandex-praktikum/java-explore-with-me/main/ewm-stats-service-spec.json/ "спецификация сервиса статистики")

## Основной сервис

Содержит всё необходимое для работы продукта.
API основного сервиса разделено на три части:
- **публичная** будет доступна без регистрации любому пользователю сети;
- **закрытая** будет доступна только авторизованным пользователям;
- **административная** — для администраторов сервиса.

Спецификация основного сервиса содержится здесь: [ewm-main-service-spec.json](https://raw.githubusercontent.com/yandex-praktikum/java-explore-with-me/main/ewm-main-service-spec.json/ "спецификация основного сервиса")

Основной сервис состоит из нескольких сервисов:

### Сервис пользователей

Данный сервис содержит функционал управления пользователями:

- добавление новых пользователей;
- удаление пользователей;
- получение существующих пользователей;
- проверка существования пользователей.

### Сервис событий

Данный сервис является ключевым и содержит основной функционал приложения.
Сервис включает в себя следующий функционал:

- Управление событиями. Добавление новых событий, изменение существующих событий и получение информации о добавленных событиях с различными отборами и с сортировкой по количеству просмотров, которое будет запрашиваться в сервисе статистики, либо по датам событий;
- Управление категориями. Каждое событие должно относиться к какой-либо категории. Имеется возможность добавлять новые категории, а также удалять и изменять существующие категории;
- Подборки событий. Должна быть настроена возможность получения всех имеющихся категорий и подборок событий (такие подборки будут составлять администраторы ресурса).

### Сервис управления заявками на участие

- настроена подача заявок на участие в интересующих мероприятиях;
- доступно изменение статуса (подтверждение, отмена) запроса на участие в событии;
- имеется возможность получения информации о заявках текущего пользователя на участие в чужих событиях;
- имеется возможность получения информации о запросах на участие в событии текущего пользователя.
